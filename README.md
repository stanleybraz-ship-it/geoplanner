# GeoPlanner

## Descrição
O GeoPlanner é uma plataforma digital de apoio ao planejamento didático para professores de Geografia,
orientada pelo desenvolvimento do raciocínio geográfico a partir de problemas do cotidiano.

## Objetivo Geral
Desenvolver uma ferramenta tecnológica que auxilie docentes na articulação entre BNCC,
metodologias ativas e avaliação formativa.

## Público-alvo
Professores de Geografia da Educação Básica.

## Fundamentação Teórica
O projeto fundamenta-se nos seguintes referenciais:
- Raciocínio Geográfico (Cavalcanti)
- Letramento Espacial (Callai)
- Design Instrucional Contextualizado (Filatro)
- Planejamento por Competências (Zabala)
## Natureza do Projeto
Projeto de pesquisa aplicada com desenvolvimento de produto tecnológico educacional,
em consonância com políticas de inovação e formação docente.

## Vínculo acadêmico
Projeto de pesquisa em Tecnologias Educacionais aplicadas ao Ensino de Geografia.
